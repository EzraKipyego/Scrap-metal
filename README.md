# Scrap Metal Website

## Introduction
This is a  web project for a scrap metal business. It helps users view services, location, and contact information for buying and selling scrap metal. It is built using HTML, CSS, and JavaScript.

## Installation
To run the project locally:

1. Clone the repository from GitHub:

```bash
git clone https://github.com/EzraKipyego/Scrap-metal.git
```
2. Navigate into the project folder:
```bash
cd Scrap-metal
```

3. Open `index.html` in your browser.
